
## 心響のひさめ

心拍数管理を主題とした2Dプラットフォームアクション

## 操作方法
| **操作** | **key** | **詳細** |
|------|------|------|
| 移動 | W A S D | 回避を行うとき W S の上下軸も使用する |
| 弱攻撃 | J 左クリック | |
| 強攻撃 | K | |
| 回復アイテム使用 | R | |
| 心拍数を抑える | L | |
| 居合切 | L | 心拍数 100 未満の時 心拍数を抑えるを解除したとき発動する |
| ガード | O |  |

## 心拍数
0 ~ 200 までの心拍数ゲージ を管理して戦闘を行う
低心拍数 高心拍数 それぞれに メリット デメリット 有

| **状態** | メリット | デメリット |
|------|------|------|
| 高心拍数 ( 心拍数 > 100 ) | 攻撃範囲拡大 / 攻撃速度加速 / 攻撃命中時多段判定化 / 攻撃力上昇 | 心拍数200到達時プレイヤキャラ がスタンする |
| 低心拍数 ( 心拍数 < 100 ) |||
