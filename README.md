## 心拍数
0 ~ 200 までの心拍数ゲージ を管理して戦闘を行う
低心拍数 高心拍数 それぞれに メリット デメリット 有

| **状態** | メリット | デメリット |
|------|------|------|
| 高心拍数 心拍数100超え時 | 攻撃範囲拡大 / 攻撃速度加速 / 攻撃命中時多段判定化 / 移動速度上昇 | 心拍数200到達時プレイヤキャラ がスタンする |
| 低心拍数 心拍数100未満時 | 居合切の攻撃力上昇 / 攻撃力上昇 | 攻撃範囲縮小 / 攻撃速度減速 / 移動速度減少 / 0 到達時スリップダメージ |

| **状態** | 条件 |
|------|------|
| 上昇条件 | 攻撃/回避を行う / 攻撃を受ける |
| 減少条件 | 心拍数を抑える / 心拍数上昇条件を達成せずに 一定時間たつと自然回復 |
